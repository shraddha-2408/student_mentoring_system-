# student_mentoring_system-

The Student Mentoring System is a web-based application designed to improve communication and interaction between students, mentors, and staff members.

## Features

- Student Registration and Login
- Admin, Staff, and Mentor Dashboards
- Student Information Management
- Academic Information Management
- Mentor Diary Module
- Student Doubt Submission and Answer System
- Meeting Management
- Parent Reporting Module
- Student Performance Tracking
- Staff Assignment and Management

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage

## Modules

- Admin Module
- Mentor Module
- Staff Module
- Student Module
- Meeting Module
- Doubt Management Module
- Academic Information Module

## Project Objective

The objective of this project is to provide an efficient platform for monitoring students' academic progress, managing mentoring activities, and facilitating communication between students and mentors.
